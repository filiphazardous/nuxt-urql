# Usage

This module reexports @urql/vue composables :
- `useQuery`
- `useMutation`
- `useSubscription`
- `useClientHandle`
- `gql`

Follow [@urql/vue documentation](https://formidable.com/open-source/urql/docs/basics/vue/) to learn more

---

## useQuery

```vue
<template>
  <div>{{ data }}</div>
</template>

<script setup lang="ts">
import { useQuery, gql } from "#imports";

const { data } = useQuery({
  query: gql`
    query name {
      # ...
    }
  `,
});
</script>
```
